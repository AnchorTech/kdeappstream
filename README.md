# kdeappstream
HTML5 backend for QT applications

Requirements
------------
* libfontconfig1-dev
* kdelibs5-dev
* libqt4-dev



Building kdeappstream on Linux\X11
----------------------------------

>$ cd ~/kdeappstream

>$ mkdir ~/kdeappstream/build

>$ cmake .. --prefix=/user/bin

>$ make

On Ubuntu/Debian
>$ sudo make install

On Fedora/Centos/Ojuba
>$ su -c make install
