# kdeappstream
HTML5 backend for QT applications

Requirements
------------
* libfontconfig1-dev
* kdelibs5-dev
* libqt4-dev
* cmake


Building kdeappstream on Linux\X11
----------------------------------

>$ cd ~/kdeappstream

>$ mkdir ~/kdeappstream/build

>$ cmake ..

>$ make

On Ubuntu/Debian
>$ sudo make install

On Fedora/Centos/Ojuba
>$ su -c make install


For Try
>$ kappstream-server

Open Location
127.0.0.1:8080
